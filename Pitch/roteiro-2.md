# Roteiro 2 — Produto

**Público:** Gestores, investidores, avaliadores de produto  
**Tom:** Foca no valor entregue — o que o usuário vê, sente e consegue fazer  
**Teclas:** `Espaço` avança elemento · `→` pula tela · `R` troca roteiro

## Página 1 — O Problema
| O que aparece na tela | O que você diz |
|---|---|
| Label "O PROBLEMA" | O problema começa aqui: toda vez que duas pessoas conversam, algo decide se vão se entender — ou não. |
| Pergunta principal em destaque | Como saber, de forma concreta, se duas pessoas **realmente se entenderam**? Essa resposta nunca existiu — até agora. |
| Linha divisória + parágrafo explicativo | Mal-entendidos custam caro: clientes perdidos, equipes em conflito, aprendizado prejudicado. O problema é real. |
| Estatística "70%" animando de 0 | **70%** dos conflitos profissionais vêm de falhas de comunicação que ninguém consegue medir com precisão. |
| Stats "3×" e "0 ferramentas" + badge TCC·IFSP·2025 | Equipes alinhadas são **3× mais produtivas**. E até hoje, zero ferramentas em português para medir isso. Este sistema muda esse cenário. |
